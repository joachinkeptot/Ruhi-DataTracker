# Tracker-

Tracker used for Bahai Activities - Expansion phase

## Overview

**Tracker-** is a visual community tracking and management system designed for Bahá'í community activities. It helps coordinators manage people, families, track spiritual education progress, organize activities, and visualize relationships between community members and programs.

### Key Features

- **People Management**: Track community members with detailed spiritual education progress (Ruhi levels)
- **Activity Tracking**: Manage Junior Youth (JY) classes, Children's Classes, Study Circles, and Devotionals
- **Family Units**: Link people to family groups for better organization
- **Geographic Mapping**: Visualize community members by area/neighborhood
- **Cohort Views**: Group people by categories (JY, Children's Classes, Youth, Parents) or family units
- **Analytics**: Track participation metrics, attendance, and engagement patterns

### Starting the Development Server

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The app will automatically open at `http://localhost:3000/index-react.html`
